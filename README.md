# V2X Poc

V2X (Vehicle-to-everything, V2X communication) Poc project implemented using Clean Architecture with MVVM design.


### Technologies & Methodologies which used:

- Koin
- Coroutines
- Clean Architecture
- MVVM Pattern
- LiveData
- Databinding
